# About

Zero is a framework for building fast, lightweight and scalable server-side applications.
It is built on top of [Dart](https://dart.dev/)

## Features

- **Fast**: Zero is built on top of Dart, which is one of the fastest programming languages out there.
- **Lightweight**: Zero is built to be lightweight, it has no dependencies and the core package is only 2.5KB.
- **Scalable**: Zero is built to be scalable, it has a built-in cluster mode that allows you to scale your application across all available CPUs.
